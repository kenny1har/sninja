ShowtimeNinja displays movie showtimes in Singapore.

This app does not consume large mobile data usage, unlike others.
Currently the cinemas are only in Singapore. Cathay, GV, and Shaw.
Still thinking to choose best movies ? Just press the TRENDING button. All popular movies will be selected.

* Quick search: Select cinema, select movie.
* Movie synopsis, duration, language
* Buy movie ticket via external web browser (secure)
* Cinema capacity indicator. Watch movie in bigger screens!
* Empty seat indicator. Choose empty cinema and get the best seat!
* Show list of movies sorted by popularity, release date
* It remembers the cinemas you have selected.
